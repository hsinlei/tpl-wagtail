## Project Detail

* Python 3
* Django 2.1.5
* Wagtail 2.8

**The database and media files would be reset after a while, so do not be surprised if your article is gone.**

## Run it in local env

```bash

# setup virtualenv
pip install -r requirements.txt

./manage.py runserver
# http://127.0.0.1:8000/blog
# username: admin  password: admin
```
